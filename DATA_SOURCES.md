# 数据源说明

## 🔒 Amazon 真实数据

**无法获取 Amazon 的真实物流数据**，原因：

1. **隐私保护**：Amazon 的物流数据包含客户信息、订单详情等敏感信息
2. **商业机密**：物流数据是 Amazon 的核心商业资产，不会公开
3. **法律限制**：未经授权访问或使用 Amazon 数据可能涉及法律问题
4. **API 限制**：Amazon 的 API（如 MWS、SP-API）需要认证，且不提供公开的物流数据接口

## ✅ 当前方案：模拟数据

我们的系统使用**智能模拟数据**，具有以下优势：

1. **完全可用**：无需依赖外部数据源，系统可以独立运行
2. **演示完整**：能够展示所有系统功能（订单、车辆、仓库、调度等）
3. **实时更新**：数据会动态变化，模拟真实的物流场景
4. **可定制**：可以根据需要调整数据量和更新频率
